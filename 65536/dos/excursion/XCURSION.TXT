  This   introduction   was   developed   during   about  two  months  before
Abduction  '96.  The  intro  takes  only  49761 bytes of  disk  space and its
playing  time is  about 4:55. I  recommend  you  have a 486/66 MHz for smooth
action. If you have a 286 you can't run this, because 386 or better is needed
(the code is faster).This intro runs either without sound or with sound using
Sound Blaster. GUS is not supported, because I don't know anybody who has got
a GUS, so I weren't able to test the player.  The intro should work with  any
memory driver (EMM386, QEMM) and  you  should have  at least 500 kilobytes of
free basememory. If the intro didn't work after all, don't  blame us, because
you're running it with your own risk.
