OPTIONS:
The options appear under a menu at the start of the intro:
        F1=LOOP (loop MODE as required by competion rules)
        F2=ONCE (once MODE aka "normal" MODE)
        F3-MOVE (move MODE ie:Interactive mode)

TROUBLESHOOTING:
Question1: "I have images but no sound/music, how come ?"
Answer1  : "DMA is hardcoded to DMA=1"
Solution1: "Change Sounblaster's config to DMA=channel#1"

Question2: "Which parts exactly are interactive ?"
Answer2  : "Scratching sequence and landscape parts"

Question3: "I cannot get the interactive part to work?"
Answer3  : "You need a PS/2 mouse and MS' MOUSE.EXE driver"
Solution3: "get a PS/2 mouse, download MOUSE.EXE, Reboot,
            press F8, at Microsoft's Dos menu, choose DOS
            bare mode, make sure MOUSE.EXE is in same
            directory as SCRATCH.COM or in %PATH% then re-
            launch SCRATCH.COM"
