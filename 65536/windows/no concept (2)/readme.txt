-----=====No Concept=====------
64KB demo by Unmanaged
Released at Evoke 2010

Defaults at 1024x768 windowed
Use commandline parameters to change it. The parameters are as follows:
-f --> for fullscreen
-r 1024x768 --> set the screen resolution (or window size) to 1024x768 (you can provide any valid resolution)

Check http://eugen.cz/noconcept for more info.
