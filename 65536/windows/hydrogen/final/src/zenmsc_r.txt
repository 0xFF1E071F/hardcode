zenith matrix script calculator

Считает матрицу по скрипту. скрипт - опкоды с параметрами, параметры
через пробел, опкоды через ";". Переводы строк - пофигу.

Расчет матрицы по сути перемножение матриц поворота и т.п. в определенном
(заданном скриптом) порядке.

опкоды:
rx <angle>; повернуть вокруг Ох на angle. angle - в ГРАДУСАХ!!!
ry <angle>; -//- Оу
rz <angle>; -//- Оz
tr <x> <y> <z>; - translate по 3 осям
sc <x> <y> <z>; - scale по 3 осям
name <name>; - задает имя матрицы (для copy packed)

когда набил скрипт - нажать Calc и Copy Packed

В клипборд пойдет упакованная матрица.
потом в своей проге paste:

char  <name>[10]={1,2,3,4,5,6,7,8,9,0}; //<----это упакованная матрица
      ^сюда пишем имя (напр packed_m), потом пишем

MATRIX m;
m.Unpack(&packed_m);
Всё, получили в m готовую к употреблению матрицу
