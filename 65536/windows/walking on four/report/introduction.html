Abstract and report

In 2005, I released Walking on Four, a 64kb intro. It came second at the bcn party, building upon the work I put in my previous work, Clothing Like A T-shirt Saying I Wish. This report describes how I expanded the way I represent musical structures, as well as created more sophisticated visual effects.
