>> GENESIS <<

 

64k intro - party version
first shown at Mekka & Symposium 2000

 

  • general information

Genesis is our first 64k intro. It started as try, how small a basic 2D/3D subsystem with DirectX/Direct3D could be made. As we noticed how small the exe actually was, we decided to make a intro out of it. Most code (the 2d-subsystem, the realtime-shadow code and the 2d-effects) was taken from a newer version of our engine Merlin, which we used in the Dark Silence demo. The 3d-engine of Genesis is based up on our new engine XDream.

Hardware requirements:
- a Pentium class processor (preferrably a Pentium II) with MMX support
- a 3d-accelerator (Voodoo or TNT are good, although it may run on older cards, too)
- Windows 95/98 with DirectX 7.0 installed

Try 32-bit rendering first. If the colors are messed, or if the demo crashed, force 16-bit rendering. The quality will be a bit lower, but you will still be able to enjoy it.

If you encounter any problems or just want to contact us, don't hesitate.
