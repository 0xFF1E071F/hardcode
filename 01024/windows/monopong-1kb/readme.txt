| [About]
|
| Small pong clone written for the 
| 2BUTTON GAME competition at dbf.
| The task was to write a game that
| must be controlled by no more than 
| 2 individual keys on a keyboard.
|
|
| [Usage]
| 
| Use left/right cursor key to 
| control.
| No additional keys for start/end
| used. The game starts right after
| it is loaded and quits automatical-
| ly after the ball gets lost.
