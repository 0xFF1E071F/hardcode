*   IMPORTANT NOTE   *

This intro does not change the screen mode, but it still depends on you to pick
the "right size" executable which matches your desktop resolution.

The graphics will look plain wrong if your desktop resolution doesn't
match the resolution mentioned in name of the executable file.

If you are in doubt, compare to the screenshot_1280x720.jpg image,
the intro looks like that after half a minute or so.
Preferred resolution is 1280 x 720.


So, to reiterate...

PREFERRED WAY TO RUN THE INTRO:

  1. Change desktop resolution to 1280x720:
    Right click (on Desktop) → Screen resolution → Resolution: 1280x720
  2. Run Synxiety_desktop_in_1280x720.exe
