  so screen resolution greatly affects performance.

  Just in case, there is a 320x240 version in the package,
  as that is a really quick little res that takes it easy
  for the gpu.

  If for any reason you want to try this prod in 320x240,
  you may need to enable (just enable, not set) that mode
  in your gpu driver -- check the gpu utility, it should
  allow insertion of custom resolutions.
