Gen Universe
.a demo in 4KB by System K ( kioku + sincl )
.Made in Japan

.released at 2ch Party 2006 Summer

必要なシステム :
- OpenGL 1.5以上をサポートしたグラフィックスカード
- 音が鳴るサウンドカード
- Windows XP SP2 以上
(注. CPUによっては事前計算で少し時間がかかる場合があります)


仕事 :
kioku - gfx,softsynth
sincl - music

thanks NPX packer (by 301)


コメント :
kiokuです。今回は64KB Introじゃなくて4KBです。
64KBもやろうとかと思いましたが、Introシステムに特に新しい機能が
増えてなかったので、同じシステムで同じようなもの作ってもなんだかなぁ
と思ったので新しいシステムを模索するためにも原点にもどって4KBにしました。
今回は64KBをつくっていた途中でわかってきたEXEを小さくするテクを含め
コンパイラが吐くアセンブラコードをみながら小さくなるようにいろいろ
やってみました。まぁ、そこそこ小さくなったんじゃないかとおもってます。
まぁ今回わかったEXEを小さくするヒントは機を見てmegademo-wikiあたりに
書こうかと思ってます。
まぁ4Kもまだいろいろできそうな気がしますね。
ではまた。
