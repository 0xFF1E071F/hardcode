This is entry  for  contests.anticrack.de's  contest for a
4kb keygen template.  The  entry is 5120 (5kb) bytes after
building (MASM), but 4kb after compressing with UPX v1.23.
The effect  is  a  classic  rotozoomer  with  an  icon  of
Shakira.
