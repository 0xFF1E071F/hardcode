Compatibility
=============
OpenGL, DX10 level graphics card. That is: Radeon HD 2x00+ or Geforce 8x00+. 
Only tested on Radeon5770 so far..


Fast party hack done with raymarching and simple bloom/dof postprocessing.
Lets hope the non-beta drivers will allow me to enable supersampling again ;)
