                      tools used:   Crinkler, Shader Minifier and Clinkster

                                .Requirements.
                                --------------
     * A fast video card with lots of Pixel Shader 4.0 compatible shader pipelines.
       The intro has been tested on AMD and NVidia cards with recent drivers.
     * 1.5 GB ram (mainly for the Crinkler decompression)
     * Developed on Windows 7, might even run on WInXP
     * 4 kb diskspace
   

                              .About the intro.
                              -----------------

This is our yearly not-so-serious christmas 4K, made in even more of a hurry than usual. All of our musicians
were swamped with real-life issuse, so a big thank you to Skyrunner/Brainstorm for mutilating a Christmas tune
as instructed. Almost all code was written on the partyplace. I aimed to get a bit more realistic christmas tree
than last year, but I'm not happy with the giant rendering artifacts. But hey, it's a compo filler :)
