*IMPORTANT*: Check that you have enabled some device for MIDI Music
             Playback if you don't get any sound! Also make sure you have
             MIDI Volume high enough.
    Control Panel->
        Sounds and Multimedia->
        Audio->
        MIDI MUsic Playback->
        Preferred device
    If possible, choose a native device for your soundcard instead of the
    "Microsoft GS Wavetable SW Synth". E.g. if you have a SB Live! sound card,
    choose "A: SB Live! MIDI Synth".
