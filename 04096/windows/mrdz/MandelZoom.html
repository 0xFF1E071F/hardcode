MRDZ
( Mandelbrot Realtime Deep Zoomer :-P )

 

I like mandelbrot pictures since a long time. That time here is a little zoomer using today hardware :). That little proggy uses two threads: the main thread computes the mandel set (using a *high* precision), and the second thread uses DirectX8 textures to smoothly zoom between pictures.

Technical details: Nothing new exept the mandelset calculation is quite optimized (recursive routine to avoid computing large flat zone). The program zoom with a total factor of 2^55 !!! (3.6e16, aproximatly the range beetween human and universe :-)) And last but not least, the EXE is only 5.5Kb long !!!

(You need to install the great DirectX8 API to run)
