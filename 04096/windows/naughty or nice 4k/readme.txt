                                .Requirements.
                                --------------
     *  A very fast video card with lots of Pixel Shader 4.0 compatible shader pipelines,
        and very recent drivers. NVidia users need the 376.33 drivers, which fix an
        internal compile error.
     * 300 MB ram (for the Crinkler decompression)
     * Tested on Windows 7 and 10, might even run on Windows XP
     * 4 kb diskspace
