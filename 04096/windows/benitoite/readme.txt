           B E N I T O I T E


A 4K intro for the Evoke 2006 4k intro compo.

This archive contains a slightly updated version of Benitoite
fixing some timing issues on slower systems, together with
a remixed version by cheesy / TBC.

Credits:

Main code and direction: Mentor & Blueberry
Synth code (kklangzeug): Gargaj / ümlaüt design
Model, keyframing, scripting: Cheesy
Music: Puryx

Requirements:

Graphics card with VS & PS 2.0 or better
DirectX 9.0c December 2005 or newer (d3dx9_28.dll)

Tools used for the development of this intro:

RenderMonkey
Lightwave
kklangzeug editor by Gargaj & tsw
Visual Studio
Emacs
grep
Various home-made tools
and of course, Crinkler

Thanks to Dipswitch for the ascii art.


P.S.: This intro is no more than a trivial application of the
D3DXCreateRockSurfaceWithGlossyCrystalsFromFileInMemory function.
