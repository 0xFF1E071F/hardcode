                     -* Description *-

     This is a 4k intro for Assembly 98 intro competition. It
     contains no sound and uses standard VGA video mode. Intro
     takes less than 550kb base memory and works under Microsoft
     Windows 95/98/NT dos box/window.

                      -* Features *-

     - Name of the intro, solid moving object
     - Fading starfield efecto, another text
     - Zooming  starfield, another object
     - Final part with multiple moving lightsources, adaptive
       surface and new 2-layer transparent object.
