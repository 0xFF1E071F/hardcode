Readme file for "spin" 4k intro for The Gathering 1998 by cortex/index.

This thing needs about 500k free conventional memory, and if you don't have
it, you have a problem.

The Adlib player is a size optimized version of RAD player, and the music
is tracked by Zauron/index using Reality ADlib Tracker.

greets go out to anyone who wants it...
