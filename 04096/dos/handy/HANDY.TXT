	Parameters:	with the '-t' or '-T' parameter the proggy
			will behave like an ordinary Tetris. (Game mode.)
			In this mode the objects's falling rate in the
			beginning is not so fast than in Intro mode.



	Without parameters you will see the ordinary intro. ESC quits
	the whole proggy (or the Tetris routine). You can play in Intro
	mode too, but if you fail the intro will be continued.



	Control keys:	Left arrow: Move object one column left.
			Right -"- : -"-   -"-   -"-   -"-  right.
			Up    -"- : Rotate " 90 degrees (clockwise).
			Backspace : New game. (Works for intro mode too).
			Escape    : Quit tetris (intro part).
			Pause     : Pause. (Restart with any other key
				    (except shift, alt, etc...)

			Down key, space, other unlisted keys increase object's
			falling rate.



	The current Tetris record is 204 (in Game mode). It was earned by me
	during testing the routines.
