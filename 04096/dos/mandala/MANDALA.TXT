  úintroductionú

	Mandala took part in the Scenest'96 4k intro competition in Hungary.
	Big surprise, the code is optimized for size not for speed, and
	maybe the syncronising is inaccurate on faster machines due to this.

	Thanks to Richard Hughes for showing to me what a mandala looks like,
	Kenneth Aastroem for the cloudgenerator, which inspired me to write
	my own in 4k, and VLad DRacl for being my English teacher.

  úrequirementsú

	You will need at least a 386 based machine with a coprocessor and VGA
	to run this 4k intro. About 530k base memory is needed for launching,
	no high memory is used. It should work fine with just about every
	memory config.
